# Analisis Penjualan Produk E-Commerce Menggunakan PySpark

Repository ini berisi proyek analisis Big Data untuk tugas UAS mata kuliah **TIF505 – Big Data**.  
Analisis dilakukan menggunakan **Apache Spark (PySpark)** dan Python untuk mengolah serta memvisualisasikan data penjualan E-Commerce.

Link ipynb:
https://nbviewer.org/github/fraxzu/Produk-E-Commerce/blob/main/product.ipynb
## Deskripsi Singkat
Proyek ini bertujuan untuk menganalisis data penjualan produk E-Commerce guna memperoleh insight seperti:
- Produk yang paling banyak terjual
- Total penjualan dan profit
- Perbandingan penjualan berdasarkan kategori dan waktu

## Dataset
Dataset yang digunakan merupakan data penjualan E-Commerce yang berisi informasi:
- Produk
- Kategori
- Jumlah terjual
- Nilai penjualan (Sales)
- Profit
- Tanggal transaksi

Dataset mencerminkan karakteristik Big Data (Volume, Variety, dan Value).

## Tools & Teknologi
- Python
- Apache Spark (PySpark)
- Jupyter Notebook
- Matplotlib & Seaborn (visualisasi)
- Streamlit / Voilà (kode dashboard)

## Alur Analisis
1. Data ingestion dan preprocessing menggunakan PySpark  
2. Analisis data penjualan dan profit  
3. Visualisasi hasil analisis  
4. Penyusunan kode dashboard berbasis Python  

## Catatan
Dashboard berbasis Python disediakan dalam bentuk kode dan dijalankan terpisah dari notebook, sesuai dengan praktik pengembangan dashboard analitik.
